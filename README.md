# CashFlow Tracker 💸

Telegram Mini App для комплексного управления личными финансами, совместными бюджетами и отслеживания финансовых целей.

## 🚀 О проекте

CashFlow Tracker — это мобильное финансовое приложение, оптимизированное для Telegram Web Apps. Оно помогает пользователям вести учет расходов и доходов, управлять долгами и кредитами, планировать финансовые цели и получать интеллектуальные рекомендации на основе истории транзакций.

Приложение ориентировано на:

- личный финансовый контроль;
- совместные бюджеты с друзьями и родственниками;
- централизованное управление регулярными платежами и подписками;
- развитие финансовой дисциплины через геймификацию;
- автоматический анализ и подсказки с помощью AI.

## 🌟 Ключевые возможности

### 1. Учет доходов и расходов

- Быстрое добавление транзакций из мобильного интерфейса.
- Поддержка категорий и субкатегорий для точного анализа.
- Сканирование чеков и автоматическое извлечение данных о покупках.
- История транзакций с фильтрами по дате, типу и категории.

### 2. Совместные бюджеты

- Совместное управление счетами и тратами.
- Приглашение участников по инвайт-коду.
- Общая статистика и распределение расходов между участниками.

### 3. Управление долгами и кредитами

- Отслеживание статусов: «Я должен» и «Мне должны».
- Поддержка графиков платежей и сроков погашения.
- Автоматический расчет ежемесячного платежа и итоговой суммы.

### 4. Финансовые цели

- Создание целей накоплений и планов для достижения.
- Визуализация прогресса в процентах.
- Автоматическое пополнение цели из доступного бюджета.

### 5. AI-аналитика (AI Insights)

- Генерация персональных рекомендаций.
- Сводки по расходам за последние 90 дней.
- Предупреждения о возможных рисках и способах сэкономить.
- Интеллектуальный анализ привычек пользователя.

### 6. Геймификация

- Стрики — серии дней с активным использованием приложения.
- Ачивки за регулярное добавление транзакций, достижение целей и экономию.
- Визуальные награды для мотивации финансовой дисциплины.

### 7. Календарь и подписки

- Планирование регулярных платежей и зарплат.
- Учет рабочих смен и дополнительного заработка.
- Отслеживание сроков подписок и их автоматическое напоминание.

## 🧩 Технологический стек

- **Frontend:** React 19, TypeScript, Vite
- **Роутинг и состояние:** TanStack Router, TanStack Query
- **Стилизация:** Tailwind CSS, Shadcn UI
- **Графики:** Recharts
- **Backend / BaaS:** Supabase (PostgreSQL, Auth, Edge Functions)
- **Интеграция:** Telegram Web Apps API
- **Пакетный менеджер:** Bun

## 📦 Установка и запуск

1. Клонируйте репозиторий:

```bash
git clone https://github.com/ВАШ_НИК/cashflow-tracker.git
cd cashflow-tracker
```

2. Установите зависимости:

```bash
bun install
```

3. Создайте файл `.env` в корне проекта и добавьте переменные Supabase:

```env
VITE_SUPABASE_URL="ваша_ссылка_supabase"
VITE_SUPABASE_PUBLISHABLE_KEY="ваш_anon_key"
```

4. Запустите проект в режиме разработки:

```bash
bun dev
```

## 🛠️ Структура проекта

- `src/` — основная frontend-логика и компоненты.
- `src/routes/` — маршруты приложения.
- `src/components/` — UI-компоненты и формы.
- `src/lib/` — утилиты, интеграции и бизнес-логика.
- `supabase/` — миграции базы данных и конфигурация.

## 💡 Рекомендации

- Для полноценной работы приложения необходимо настроить проект в Supabase и подключить Telegram Web Apps.
- Проверьте правильность `VITE_SUPABASE_URL` и `VITE_SUPABASE_PUBLISHABLE_KEY`.
- При необходимости добавьте дополнительные ключи и секреты в файл `.env`.
